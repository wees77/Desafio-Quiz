# Desafio Quiz App

## Aplicativo Android desenvolvido em Kotlin utilizando Jetpack Compose, com o objetivo de apresentar um quiz interativo com perguntas e pontuação ao final.

# Funcionalidades
* Tela de login inicial
* Navegação entre telas com NavController
* Exibição de perguntas com múltiplas opções
* Avanço automático ao responder
* Cálculo de pontuação
* Tela final com resultado
* Opção de jogar novamente
